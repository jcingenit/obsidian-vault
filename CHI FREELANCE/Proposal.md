# 13. Proposal structure

Every proposal should be simple.

## Proposal sections

1. Client goal
2. Current website problem
3. Recommended solution
4. Scope of work
5. Timeline
6. Investment
7. What is included
8. What is not included
9. Payment terms
10. Next steps

### Example proposal language

**Goal:**  
Help [Business Name] create a cleaner, more mobile-friendly website that makes it easier for potential customers to understand the business, trust the brand, and take action.

**Recommended solution:**  
A refreshed 5-page website with clearer messaging, stronger calls-to-action, improved mobile layout, and basic local SEO setup.

**Investment:**  
$2,250 total  
50% deposit to begin  
25% after first draft  
25% before launch

**Timeline:**  
2–4 weeks depending on content and feedback.
